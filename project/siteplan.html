Site Purpose

The website will provide the following:

  • An interactive menu with filtering capabilities
  • Online ordering system for pickup/delivery
  • Gallery of signature creations
  • Information about baking classes and catering
  • Customer testimonials and reviews

Scenrios

Our primary visitors will be:

  1. Wedding Planners

    "What gluten-free wedding cake options do you offer that can withstand outdoor summer temperatures?"

    • Solution: Filterable menu with "wedding" and "gluten-free" tags
    • Technical: Array filtering and conditional rendering
  2. International Customers

    "I live abroad and want to send a cake to my mum in Lagos for her birthday"

    • Solution: International delivery flow with currency conversion
    • Technical: API integration for exchange rates and localStorage for draft orders
  3. Local Office Managers

    "How can I place a recurring order for weekly staff meetings with custom pastry selections?"

    • Solution: Account-free reorder system using localStorage
    • Technical: Cart persistence and template literals for order summaries
  4. Parents Planning Event

    "Can I order matching themed cupcakes and cake for my child's birthday party next month?"

    • Solution: Party package builder with date validation
    • Technical: Date comparison logic and DOM manipulation for package customization
